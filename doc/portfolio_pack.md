# Portfolio Pack

## GitHub repository description

`Desktop IoT Orchestrator` is a desktop control suite for MQTT systems built with PySide6, FastAPI, and vanilla HTML/CSS/JavaScript.
It includes a startup launcher, dependency checks, a modern dashboard, and a clean structure designed for portfolio presentation.

## Short GitHub description

Desktop IoT control suite with a PySide6 launcher, FastAPI backend, and MQTT dashboard.

## Longer GitHub description

Desktop IoT Orchestrator is a portfolio-ready desktop application for controlling and testing MQTT workflows.
It combines a PySide6 launcher, automatic dependency checks, a FastAPI control backend, and a modern SaaS-style dashboard.
