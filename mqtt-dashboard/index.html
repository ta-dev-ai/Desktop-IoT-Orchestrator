Commandes disponibles

Repère simple pour débuter et modifier le backend plus tard.

# Commande Description Action
1 mosquitto -v Démarrer le broker MQTT
2 mosquitto_sub -h localhost -t temperature S'abonner au topic temperature
3 mosquitto_pub -h localhost -t temperature -m "25°C" Publier température 25°C
4 netstat -an | findstr :1883 Vérifier le port 1883
